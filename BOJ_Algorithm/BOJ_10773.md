### BOJ 10773번 풀이
https://www.acmicpc.net/problem/10773

파이썬 풀이
```
import sys

input = sys.stdin.readline

count = int(input())
num_lst = []

for i in range(count):
    num = int(input())
    if num == 0:
        num_lst.pop()
    else :
        num_lst.append(num)

total = sum(num_lst)

print(total)
```

자바 풀이(파이썬으로 푼 후 AI를 통한 java로 변경)
```
import java.io.BufferedReader;
import java.io.InputStreamReader;
import java.util.ArrayDeque;
import java.util.Deque;

public class Main {
    public static void main(String[] args) throws Exception {
        // 1. 빠른 입력을 위한 BufferedReader
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        
        // 2. 명령어 개수 K 입력
        int K = Integer.parseInt(br.readLine());
        
        // 3. 숫자를 담을 스택 (ArrayDeque 사용)
        Deque<Integer> stack = new ArrayDeque<>();
        
        for (int i = 0; i < K; i++) {
            int num = Integer.parseInt(br.readLine());
            
            if (num == 0) {
                // 4. 0이 입력되면 가장 최근 숫자 제거 (pop)
                // 문제 조건상 비어있을 때 0이 들어오는 경우는 없으므로 바로 pop 가능
                stack.pop();
            } else {
                // 5. 0이 아니면 스택에 추가 (push)
                stack.push(num);
            }
        }
        
        // 6. 스택에 남아있는 모든 숫자의 합 구하기
        long totalSum = 0;
        // 스택이 빌 때까지 하나씩 꺼내며 더함
        while (!stack.isEmpty()) {
            totalSum += stack.pop();
        }
        
        // 7. 결과 출력
        System.out.println(totalSum);
    }
}
```
